# CHRISTMAS-PACKAGE
## SIMPLE MODULE
- helloworld() --> prints hello world
- helloname(myname) --> prints hello with a given name
- sum(x, y) --> Calculates the sum of two given numbers
- average(x, y) --> Calculates the average of two given numbers
- power(x, y) --> Calculates the power of one number to the other
- evenOrOdd(x) --> checks if a given number is even or odd

## AnthonysLoveJourney
- calculateOldLove(name1, name2) --> returns a love % between 2 given names
- calculateTrueLove(name1, name2) --> return a true love percentage
